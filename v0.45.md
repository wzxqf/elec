# elec 项目 v0.45 发布说明

## 1. 版本定位

`v0.45` 是当前唯一正式口径，对应：

- `project.version: v0.45`
- `training.algorithm: HYBRID_PSO_V040`
- `outputs/v0.45/`

本轮重点不是扩展新机制，而是把项目文档、配置、运行链、测试和版本化产物统一到当前正式实现。

## 2. 当前正式参数

当前正式参数以根配置为准：

- 上层 `contract_curve_latent.size = 20`
- 上层 `action_head.size = 8`
- 下层四组响应 `response_size = 12`
- HPSO 优化器 `init_scale/inertia/cognitive/social/position_clip_abs = 0.12 / 0.68 / 1.30 / 1.45 / 1.25`
- 上层搜索预算 `192 x 320`，真实维度 `186`
- 下层搜索预算 `160 x 320`，真实维度 `48`

## 3. 文档与仓库治理

当前活跃文档已统一到正式主线：

- `README.md`：当前入口、运行方式、输出结构
- `docs/v0.45_architecture_implementation.md`：详细架构与实现机理
- `docs/ARCHITECTURE.md`：架构索引
- `docs/CONSTRAINTS.md`：当前边界
- `docs/STATE_SCHEMA.md`：当前状态结构
- `docs/agents.md`：策略层说明

历史规划、历史规格、历史试验说明均已迁入 `已归档/`，不再作为当前项目说明的一部分。

## 4. 当前输出口径

正式运行后应生成：

- `outputs/v0.45/logs/`
- `outputs/v0.45/models/`
- `outputs/v0.45/metrics/`
- `outputs/v0.45/reports/`

其中版本总报告统一命名为 `v0.45报告.md`。

推荐验证命令：

python -m src.scripts.run_pytest tests/test_v045_hpso_param_config.py tests/test_v045_repo_cleanup.py tests/test_v045_run_all_script.py tests/test_v045_reporting_contracts.py tests/test_v045_versioning.py -q

## 6. 结论

`v0.45` 当前已经完成活跃文档重构、历史说明归档、运行链口径收口和仓库清理。后续所有正式开发、测试和产物导出均应继续围绕 `experiment_config.yaml + HYBRID_PSO_V040 + outputs/v0.45/` 展开。
