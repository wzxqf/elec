# 总规划（v0.45正式口径）

## 一、课题定位

### 1. 论文题目

**湖南电力市场售电公司多时间尺度采购策略优化研究——基于双层混合粒子群搜索与政策可行域约束**

### 2. 研究主体与研究场景

本文以湖南省电力市场中的售电公司为研究主体，围绕售电公司在批发侧购电、零售侧售电以及现货市场风险暴露管理中的决策问题展开。研究场景聚焦于中长期合约持仓、现货边际修正和结算风险控制之间的联动关系，核心关注售电公司如何在地方市场规则约束下形成可执行、可评估、可解释的采购策略。

### 3. 研究时间范围

研究样本期为：

\[
2025\text{年}11\text{月}1\text{日}00{:}00 \;\sim\; 2026\text{年}3\text{月}20\text{日}23{:}45
\]

该时间范围覆盖湖南电力现货规则、中长期交易规则、新能源机制电价竞价安排以及2026年度中长期价格联动机制进入研究样本的关键阶段，适合观察制度状态变化下售电公司采购策略的调整逻辑。

### 4. 项目实现口径

论文实证部分以项目 `v0.45` 为正式实现口径。项目主链路为：

```text
data -> policy -> feasible_domain -> tensor_bundle -> HYBRID_PSO_V040 -> materialize/evaluate/backtest -> reporting
```

对应到论文中，项目不是单纯的算法实验，而是一个完整的“制度识别—状态构造—策略搜索—约束投影—结算回测—结果审计”框架。论文写作应围绕这一框架展开，避免把模型解释成单一收益最大化工具。

---

## 二、核心研究问题

本文拟回答三个问题。

第一，在湖南电力市场中长期合约仅作为结算与风险管理依据、现货市场承担边际价格形成职能的制度环境下，售电公司如何确定周度中长期持仓规模、持仓调整量和风险敞口带宽。

第二，在既定周度中长期持仓基础上，售电公司如何根据价差、负荷预测偏差、新能源扰动和制度状态，对小时级现货边际修正进行参数化决策，并通过15分钟代理结算衡量其成本与风险后果。

第三，将政策规则编译为可行域边界后，双层混合粒子群搜索模型能否在收益表现、尾部风险、套保误差和策略可执行性之间形成相对均衡的决策结果。

## 三、研究边界

### 1. 市场边界

本文纳入以下市场环节：

- 中长期电能量交易；
- 日前现货市场；
- 日内/实时现货结算代理；
- 零售侧收入边界；
- 与主策略有关的政策状态与制度约束。

辅助服务、需求响应、虚拟电厂、储能独立优化等内容仅作为制度环境或状态变量背景处理，不扩展为独立收益来源或独立决策主体。

### 2. 决策边界

售电公司的决策被划分为三个时间尺度：

| 时间尺度 | 决策含义 | 论文中的作用 |
|---|---|---|
| 周度 | 中长期合约目标持仓、持仓调整量、风险敞口带宽 | 构成上层策略搜索对象 |
| 小时级 | 现货边际修正方向与规模 | 构成下层策略搜索对象 |
| 15分钟 | 代理结算、偏差成本、尾部风险计算 | 构成回测评价层 |

### 3. 收益边界

本文不构造完整财务利润表。零售侧收入按外生零售电价参数处理，研究重点放在批发采购成本、现货偏差风险、交易摩擦、政策投影惩罚以及相对基准策略的超额表现上。

#### 1.1 研究背景与意义

从全国统一电力市场建设、湖南电力现货市场推进、新能源市场化改革和售电公司风险暴露变化四个层面展开。重点说明售电公司不再只面对单一合约采购问题，而是需要同时处理中长期持仓、现货价格波动、负荷偏差和制度时点变化。

#### 1.2 国内外研究现状

围绕以下方向梳理文献：

- 电力中长期合约与差价合约机制；
- 售电公司购售电决策与风险管理；
- 电力现货市场与中长期衔接；
- 可再生能源消纳与新能源市场化机制；
- 粒子群优化、混合智能优化及其在能源系统中的应用；
- CVaR、尾部风险和组合风险评价。

文献综述应服务于本文方法选择。重点不是证明某一算法先进，而是说明售电公司采购决策具有多时间尺度、非线性、强约束和多目标评价特征，适合采用参数化搜索与制度约束结合的建模方式。

#### 1.3 研究问题与研究思路

明确本文从湖南市场制度出发，将售电公司的采购行为抽象为周度中长期持仓与小时级现货修正的联合决策问题，并通过15分钟代理结算检验策略后果。

#### 1.4 研究内容、技术路线与创新点

研究内容包括制度机制识别、风险因子提取、数学模型构建、双层HPSO求解、滚动回测评价和策略解释。创新点应控制在三个方面：

- 将湖南电力市场政策规则编译为策略可行域，而不是仅作为背景文字；
- 构建周度—小时级—15分钟的多时间尺度采购策略评价框架；
- 通过双层HPSO搜索中长期持仓和现货边际修正参数，并结合基准比较、消融和稳健性检验评价模型表现。

### 第二章 湖南电力市场制度基础与售电公司风险机理

#### 2.1 湖南电力市场交易结构

说明湖南电力市场中发电企业、售电公司、电力用户、电网企业和市场运营机构的基本关系。重点解释售电公司在批发市场与零售市场之间承担的中介性风险。

#### 2.2 中长期交易与现货市场衔接机制

中长期合约在本文中被处理为结算依据和风险管理工具，现货市场承担全电量集中竞价和边际价格形成职能。论文应解释为什么中长期持仓不能被理解为物理调度计划，也不能简单等同于实际购电执行量。

#### 2.3 零售套餐与收益端边界

零售侧以套餐或合同方式锁定用户销售关系，本文将其作为收入端边界处理。模型不把零售套餐设计作为自由动作，而是围绕采购侧成本和风险控制展开。

#### 2.4 新能源机制电价与价格联动制度

对新能源机制电价竞价、机制电量比例、机制电价上下限、执行期限等制度安排进行梳理。对于2026年度中长期价格机制，应重点说明固定价格比例与实时联动比例对中长期合约风险管理属性的影响。

#### 2.5 售电公司风险来源

售电公司面临的主要风险包括：

- 中长期持仓与实际净需求不匹配；
- 日前与日内价格波动；
- 负荷预测偏差；
- 新能源出力扰动；
- 交易摩擦和偏差结算成本；
- 政策规则切换带来的结算口径变化。

### 第三章 售电公司多时间尺度采购策略建模

#### 3.1 符号定义

设：

- \(w \in \mathcal{W}\)：周度索引；
- \(h \in \mathcal{H}_w\)：周内小时索引；
- \(t \in \mathcal{T}_w\)：周内15分钟结算时段索引；
- \(\widehat{L}_w\)：第 \(w\) 周预测净需求电量；
- \(L_w\)：第 \(w\) 周实际净需求电量；
- \(p^{DA}_{w,t}\)：日前价格；
- \(p^{ID}_{w,t}\)：日内/实时代理价格；
- \(p^{LT}_{w}\)：中长期结算代理价格；
- \(\tau\)：零售侧单位收入参数；
- \(Q^{LT}_w\)：第 \(w\) 周中长期持仓；
- \(a^{raw}_w\)：模型生成的原始中长期调整量；
- \(a_w\)：经过可行域投影后的中长期调整量；
- \(B_w\)：第 \(w\) 周允许的现货边际敞口带宽；
- \(u_{w,h}\)：第 \(w\) 周第 \(h\) 小时现货边际修正量。

#### 3.2 周度中长期持仓模型

上层模型先生成原始中长期调整量：

\[
a^{raw}_w
= \eta_a \tanh\left(\theta_x^\top X_w + \theta_s^\top S_w + b_a\right)\widehat{L}_w
\]

其中，\(X_w\) 表示周度市场特征，\(S_w\) 表示政策状态特征，\(\theta_x\)、\(\theta_s\) 为待搜索参数，\(\eta_a\) 为调整规模系数。

政策可行域给出中长期调整比例边界：

\[
\underline{r}^{a}_w \leq \frac{a_w}{\widehat{L}_w} \leq \overline{r}^{a}_w
\]

因此，执行调整量为：

\[
a_w
= \Pi_{[\underline{r}^{a}_w \widehat{L}_w,\,\overline{r}^{a}_w \widehat{L}_w]}\left(a^{raw}_w\right)
\]

周度中长期持仓为：

\[
Q^{LT}_w
= \max\left(0,\rho_0\widehat{L}_w + a_w\right)
\]

其中，\(\rho_0\) 为基础持仓比例。非负约束体现售电公司采购侧中长期净持仓的执行边界。

#### 3.3 风险敞口带宽模型

模型生成原始现货边际敞口带宽：

\[
B^{raw}_w
= \eta_B \widehat{L}_w\left[1+\tanh\left(\theta_B^\top X_w+b_B\right)\right]
\]

可行域给出带宽比例边界：

\[
\underline{r}^{B}_w \leq \frac{B_w}{\widehat{L}_w} \leq \overline{r}^{B}_w
\]

执行带宽为：

\[
B_w
= \Pi_{[\underline{r}^{B}_w\widehat{L}_w,\,\overline{r}^{B}_w\widehat{L}_w]}\left(B^{raw}_w\right)
\]

\(B_w\) 决定下层小时级现货修正的总边界。它不是收益指标，而是风险预算和策略活动空间。

#### 3.4 24小时中长期合约曲线

设 \(z_w\) 为上层粒子中的合约曲线隐变量，\(\phi_h\) 为24小时基函数，则第 \(h\) 小时的合约分解权重为：

\[
c_{w,h}
= \frac{\exp(z_w^\top \phi_h)}{\sum_{k=1}^{24}\exp(z_w^\top \phi_k)}
\]

满足：

\[
\sum_{h=1}^{24}c_{w,h}=1,\qquad c_{w,h}\geq 0
\]

周度中长期持仓可按24小时曲线分解：

\[
q^{LT}_{w,h}=Q^{LT}_w c_{w,h}
\]

在15分钟结算层中，小时级持仓进一步映射到相应的15分钟时段。

#### 3.5 小时级现货边际修正模型

下层模型根据价差、负荷偏差、新能源扰动和时段状态形成小时级信号：

\[
s_{w,h}
= \beta_1 spread_{w,h}
+ \beta_2 dev^{load}_{w,h}
+ \beta_3 dev^{ren}_{w,h}
+ \beta_4 session_{w,h}
+ \beta_5 policy_{w,h}
\]

原始现货修正量为：

\[
u^{raw}_{w,h}
= \tanh(s_{w,h}) \cdot \overline{u}_{w,h}
\]

其中，\(\overline{u}_{w,h}\) 由周度敞口带宽和小时级政策边界决定：

\[
\overline{u}_{w,h}
= \frac{B_w}{|\mathcal{H}_w|}\cdot \gamma_{w,h}
\]

\(\gamma_{w,h}\) 为小时级现货修正份额上限。若制度状态收紧，\(\gamma_{w,h}\) 相应下降。

执行修正量经过小时级投影：

\[
u_{w,h}
= \Pi_{hour}\left(u^{raw}_{w,h};B_w,\gamma_{w,h},\delta_{w,h}\right)
\]

其中，\(\delta_{w,h}\) 表示小时级爬坡约束。该投影同时控制单小时修正规模和相邻小时修正变化幅度。

#### 3.6 15分钟代理结算模型

设第 \(t\) 个15分钟时段的计划电量为 \(q^{sch}_{w,t}\)，实际需求为 \(q^{real}_{w,t}\)。计划电量由中长期分解电量和现货边际修正共同形成：

\[
q^{sch}_{w,t}=q^{LT}_{w,t}+q^{spot}_{w,t}
\]

采购成本为：

\[
C_{w,t}
= q^{sch}_{w,t}\left(\omega_{LT}p^{LT}_{w}+\omega_{DA}p^{DA}_{w,t}\right)
+ \left|q^{real}_{w,t}-q^{sch}_{w,t}\right|p^{ID}_{w,t}\mu
\]

其中，\(\omega_{LT}\) 和 \(\omega_{DA}\) 为中长期与日前代理价格权重，\(\mu\) 为偏差惩罚倍率。

第 \(w\) 周总采购成本为：

\[
C_w=\sum_{t\in\mathcal{T}_w}C_{w,t}
\]

零售侧收入为：

\[
R_w=\tau L_w
\]

交易摩擦成本为：

\[
F_w=\kappa_f\sum_{h\in\mathcal{H}_w}|u_{w,h}|
\]

投影调整成本为：

\[
A_w=\kappa_a |a^{raw}_w-a_w|
\]

周度利润为：

\[
\Pi_w=R_w-C_w-A_w-F_w
\]

#### 3.7 尾部风险与套保误差

对于第 \(w\) 周15分钟成本序列 \(\{C_{w,t}\}_{t\in\mathcal{T}_w}\)，设置信心水平 \(\alpha=0.99\)，其尾部风险指标为：

\[
VaR_{\alpha,w}=\inf\{c:P(C_{w,t}\leq c)\geq \alpha\}
\]

\[
CVaR_{\alpha,w}
= \mathbb{E}\left[C_{w,t}\mid C_{w,t}\geq VaR_{\alpha,w}\right]
\]

在样本实现中，可用尾部样本均值近似：

\[
CVaR_{\alpha,w}
=\frac{1}{|\mathcal{T}^{tail}_w|}\sum_{t\in\mathcal{T}^{tail}_w} C_{w,t}
\]

其中：

\[
\mathcal{T}^{tail}_w=\{t:C_{w,t}\geq VaR_{\alpha,w}\}
\]

套保误差定义为计划采购与实际需求的相对偏差：

\[
HE_w=\frac{|L_w-Q^{sch}_w|}{\max(L_w,\epsilon)}
\]

其中，\(Q^{sch}_w\) 为周度计划采购电量，\(\epsilon\) 为数值稳定项。

#### 3.8 政策投影惩罚

政策投影产生的可行域裁剪缺口定义为：

\[
G_w
= |a^{raw}_w-a_w|+|B^{raw}_w-B_w|+\sum_{h\in\mathcal{H}_w}|u^{raw}_{w,h}-u_{w,h}|
\]

政策违规惩罚为：

\[
V_w=\lambda_V G_w
\]

该指标用于区分“策略原始意图”和“制度可执行结果”，并为后续约束激活分析提供依据。

#### 3.9 基准策略与超额收益

设基准策略利润为 \(\Pi^{base}_w\)，则模型相对基准的超额利润为：

\[
\Delta \Pi_w=\Pi_w-\Pi^{base}_w
\]

本文保留多类基准策略，包括动态锁定基准、固定持仓基准、无现货修正基准和简单滚动对冲基准。基准比较用于检验模型在不同评价口径下的相对表现。

#### 3.10 训练奖励函数

模型训练目标不是单纯最大化利润，而是在超额收益、尾部风险、套保误差、交易摩擦和政策可执行性之间进行权衡。周度奖励函数为：

\[
r_w
=\Delta \Pi_w
-\lambda_{tail}CVaR_{0.99,w}
-\lambda_{hedge}HE_w
-\lambda_{trade}F_w
-\lambda_{violate}V_w
\]

总体优化目标为：

\[
\max_{\Theta^U,\Theta^L}\sum_{w\in\mathcal{W}_{train}} r_w
\]

等价地，项目评分核采用最小化负奖励的形式：

\[
\min_{\Theta^U,\Theta^L} -\sum_{w\in\mathcal{W}_{train}} r_w
\]

### 第四章 双层混合粒子群搜索模型

#### 4.1 参数结构

模型参数分为上层参数 \(\Theta^U\) 和下层参数 \(\Theta^L\)。

上层参数负责：

- 周度市场特征权重；
- 政策状态特征权重；
- 中长期合约曲线隐变量；
- 周度动作头参数。

下层参数负责：

- 价差响应；
- 负荷偏差响应；
- 新能源扰动响应；
- 政策收缩响应。

#### 4.2 双层粒子表示

设上层粒子为 \(x_i^{U,k}\)，下层粒子为 \(x_j^{L,k}\)，其中 \(k\) 表示迭代轮次。任意一组上层粒子和下层粒子共同形成一个完整策略：

\[
\Theta_{ij}^{k}=\left(x_i^{U,k},x_j^{L,k}\right)
\]

评分核对所有上层—下层粒子组合进行批量评价，得到：

\[
S_{ij}^{k}=J\left(x_i^{U,k},x_j^{L,k}\right)
\]

其中，\(J\) 为负奖励目标值。

#### 4.3 粒子速度与位置更新

对任一粒子，其速度更新为：

\[
v_i^{k+1}
=\omega v_i^{k}
+c_1 r_1\odot(p_i^{k}-x_i^{k})
+c_2 r_2\odot(g^{k}-x_i^{k})
\]

位置更新为：

\[
x_i^{k+1}=x_i^k+v_i^{k+1}
\]

并执行参数裁剪：

\[
x_i^{k+1}=\Pi_{[-M,M]}(x_i^{k+1})
\]

其中，\(\omega\) 为惯性权重，\(c_1\) 和 \(c_2\) 分别为个体学习因子和群体学习因子，\(r_1,r_2\) 为随机扰动项，\(p_i^k\) 为个体历史最优位置，\(g^k\) 为全局最优位置，\(M\) 为参数绝对值上限。

#### 4.4 上下层协同评价

上层粒子的个体表现由其在所有下层粒子组合中的最优得分确定：

\[
S_i^{U,k}=\min_j S_{ij}^{k}
\]

下层粒子的个体表现由其在所有上层粒子组合中的最优得分确定：

\[
S_j^{L,k}=\min_i S_{ij}^{k}
\]

全局最优组合为：

\[
(i^*,j^*)=\arg\min_{i,j}S_{ij}^{k}
\]

最终输出的策略参数为：

\[
\Theta^*=\left(x_{i^*}^{U},x_{j^*}^{L}\right)
\]

#### 4.5 训练、验证与回测

模型训练采用训练集周度样本搜索参数，在验证集物化执行轨迹，并在测试期和滚动窗口中评价策略表现。滚动回测采用扩展窗或滚动窗方式反复重训，以检验策略在不同市场阶段中的稳定性。

## 五、数据与变量体系

### 1. 原始数据

原始数据来自 `total.csv`，频率为15分钟。数据处理后形成三类样本：

- 周度样本：用于上层持仓决策；
- 小时级样本：用于下层现货修正；
- 15分钟样本：用于结算回测和风险评价。

### 2. 周度特征

周度特征包括：

- 上一周价差均值；
- 日前价格均值；
- 日内价格均值；
- 负荷偏差波动；
- 新能源比例或新能源扰动指标；
- 制度状态变量；
- 政策事件前瞻倒计时变量。

### 3. 小时级特征

小时级特征包括：

- 日前—日内价差；
- 负荷预测偏差；
- 新能源扰动；
- 峰、谷、营业时段标记；
- 价格有效性标记；
- 结算有效性标记；
- 辅助服务或制度状态收缩标记。

### 4. 政策状态变量

政策状态变量包括：

- 中长期结算基础状态；
- 现货边际敞口状态；
- 中长期—现货耦合状态；
- 新能源机制电价执行状态；
- 中长期价格联动状态；
- 调峰辅助服务暂停状态；
- 调频容量耦合状态；
- 计量和信息披露边界状态。

这些变量共同进入政策可行域和策略评分过程。

## 六、政策可行域与制度映射

### 1. 可行域编译逻辑

政策规则不只用于文字背景，而是被转化为动作边界。可行域包括：

\[
\mathcal{D}_w=\{\underline{r}^{a}_w,\overline{r}^{a}_w,\underline{r}^{B}_w,\overline{r}^{B}_w,\gamma_{w,h},\delta_{w,h},m_w\}
\]

其中，\(m_w\) 表示结算语义，如中长期价格代理方式。

### 2. 制度时点

本文采用以下制度时点：

| 制度事项 | 进入模型的方式 |
|---|---|
| 中长期合约作为结算依据 | 作为周度持仓和15分钟结算的基础约束 |
| 现货市场96点出清与结算 | 作为15分钟代理结算层约束 |
| 新能源机制电价执行 | 作为新能源机制状态变量和前瞻变量 |
| 2026年度中长期价格联动 | 作为中长期结算代理价格切换变量 |
| 调峰辅助服务暂停与调频市场耦合 | 作为现货修正带宽和政策收缩状态 |
| 计量数据与信息披露要求 | 作为数据质量和结算可追踪性约束 |

### 3. 可行域投影含义

若模型生成的动作超出制度或风险预算边界，则通过投影函数修正为可执行动作。该过程保留投影缺口，用于度量模型原始策略与现实约束之间的距离。

## 七、实证设计

### 1. 样本划分

样本按周划分为训练集、验证集和测试集。训练集用于搜索策略参数，验证集用于观察模型稳定性，测试集和滚动窗口用于评价泛化表现。

### 2. 主策略

主策略为双层HPSO策略。其输出包括：

- 周度中长期调整量；
- 周度中长期持仓；
- 周度现货敞口带宽；
- 24小时合约曲线；
- 小时级现货边际修正；
- 15分钟代理结算结果。

### 3. 基准策略

设置以下基准：

- 动态锁定基准；
- 固定持仓比例基准；
- 无现货修正基准；
- 简单滚动对冲基准。

基准策略不是陪衬，而是用于识别模型在不同市场状态下的真实优势和不足。

### 4. 消融实验

消融实验用于评价不同模块的边际贡献，主要包括：

- 去除政策投影；
- 去除状态增强；
- 去除参数布局增强；
- 保留完整模型。

### 5. 稳健性检验

稳健性检验围绕以下情景展开：

- 中长期持仓比例扰动；
- 政策切换时点扰动；
- 预测误差比例扰动；
- 市场波动强度变化。

## 八、结果分析写作口径

### 1. 收益表现

收益分析应同时报告累计利润、平均周度利润、正收益窗口数、负收益窗口数和相对基准差异。若模型在某些口径下未取得最高利润，应如实解释其原因。

### 2. 风险控制

风险分析重点关注 CVaR99、套保误差、采购成本波动和最大回撤。模型的价值不只体现在利润水平，也体现在尾部风险和偏差暴露控制上。

### 3. 政策可执行性

政策可执行性分析应基于可行域边界、投影缺口和约束激活报告展开。若运行期主要触发默认带宽边界，应按实际结果说明，而不夸大政策收紧触发频率。

### 4. 基准比较

基准比较应区分利润领先、风险领先和综合表现。若静态策略在持出集利润上更高，应解释为样本阶段内现货边际修正的收益空间有限，不能将主模型表述为在所有指标上占优。

### 5. 消融与稳健性

消融结果用于说明政策投影、状态变量和参数布局是否具有实际贡献。稳健性结果用于判断模型是否对预测误差、持仓比例和政策时点过度敏感。

## 十、论文预期结论

本文预期形成以下结论。

第一，湖南电力市场中长期合约与现货市场之间的关系已经表现出明显的结算型、风险管理型特征。售电公司需要同时管理中长期持仓、现货边际采购和实际负荷偏差，单一合约比例策略难以完整刻画其风险暴露。

第二，将政策规则编译为可行域边界，有助于把制度约束嵌入模型训练和策略物化过程。与只在文字层面描述政策不同，可行域投影能够记录策略动作被制度边界修正的程度，从而增强模型结果的可解释性。

第三，双层HPSO能够在周度持仓和小时级现货修正之间形成统一的参数化搜索框架。上层粒子决定中长期底仓和敞口预算，下层粒子决定现货修正响应，两者通过15分钟结算层共同接受收益与风险评价。

第四，模型效果应从收益、风险和可执行性三个维度综合判断。若某些静态基准在局部样本期内获得更高利润，并不否定主模型的研究价值，但说明现货边际修正的收益依赖市场波动状态、交易摩擦和偏差成本环境。

第五，从管理启示看，售电公司应避免单纯追求更高现货修正频率，而应在合约底仓、风险预算和现货边际调整之间建立动态平衡。对于政策制定和市场运营而言，更稳定的信息披露、计量数据质量和价格联动机制有助于降低售电公司策略不确定性。
